LapTime is an AppleScript library to measure execution time of a script of AppleScript. It is useful to evaluate performance of the script and to tune up the script.

You can measure execution time by using current date command which is AppleScript's built-in command. But time resolution of AppleScript's date class is 1 sec, which is too rough for performance measurements. The time resolution of LapTime is under a several msec.

You can obtain time differences bettween commands of LapTime as follows. The simple usage is using start_timer and duration. start_timer generate a new timer object. duration command obtain elapsed time from start_timer command.

record_time method records the called timing in the LapTime instance.Elapsed times between record_time can be obtained by lap_times.

use LapTime : script "LapTime"

set tm to LapTime's start_timer()
tm's record_time() -- record the timming
delay 0.1
tm's record_time()
delay 0.2
tm's record_time()

-- pretty print elapsed times between record_time
tm's lap_times()
(*[LapTime records]
8.116960525513 [ms]
112.017035484314 [ms]
210.969924926758 [ms]*)

-- obtain tatal of lap times
tm's total_time() --331.103920936584 [ms]

-- obtain elapsed time form start_timer()
tm's duration() --372.91693687439 [ms]

Constructor

start_timer

Make an instance of LapTime

Instance Methods

duration

Return elapsed time form start_timer in msec

record_time

Record elapsed time form start_timer in the instance

lap_times

Return time difference in msec between times recorded with record_time

total_time

Return time difference between start_timer and last record_time in msec
